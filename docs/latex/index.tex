Ce projet est une implémentation d\textquotesingle{}un shoot\textquotesingle{}em up inspiré de R-\/\+Type, utilisant un \doxylink{classECS}{ECS} (Entity Component System) pour la game engine côté client, et un serveur UDP pour le réseau.\hypertarget{index_autotoc_md1}{}\doxysubsection{\texorpdfstring{Aperçu Général}{Aperçu Général}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Client}} \+: Gère le rendering (SFML), inputs, mouvements, collisions, etc. via \doxylink{classECS}{ECS}.
\item {\bfseries{Serveur}} \+: Gère le réseau (UDP), interprétation de paquets, et boucle de jeu.
\item {\bfseries{Architecture}} \+: Basée sur \doxylink{classECS}{ECS} pour flexibilité ; voir la hiérarchie des classes pour les systèmes héritant de \doxylink{classISystem}{ISystem}.
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Sections Principales}{Sections Principales}}\label{index_autotoc_md2}
