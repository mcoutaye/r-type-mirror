\doxysubsection{Description détaillée}
Client TCP pour l\textquotesingle{}établissement de connexion et keep-\/alive. 

Responsable de \+:
\begin{DoxyItemize}
\item Connexion initiale au serveur TCP
\item Envoi régulier de "{}ping"{}
\item Réception de "{}pong"{} pour détecter les déconnexions
\end{DoxyItemize}

Utilisé en parallèle du canal UDP pour le gameplay.
