Description détaillée

template<typename T>
class Nwk::SafeQueue< T >

File d'attente thread-safe générique.

Paramètres du template
TType des éléments stockés dans la file

Utilisée pour communiquer entre les threads réseau et la logique du jeu (ex: inputs reçus, paquets à envoyer, snapshots d'entités). Protégée par un mutex pour un accès concurrentiel sécurisé.
