Detailed Description

template<typename ComponentType>
class StorageComponent< ComponentType >

Manages storage for a specific type of component.

This class stores components of the same type for all entities and uses a bitset to track which entities have this component.

Template Parameters
ComponentTypeThe type of component being stored.
