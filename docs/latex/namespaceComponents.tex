\doxysubsection{Description détaillée}
Espace dédié pour tous les composants \doxylink{classECS}{ECS}. 

Ce namespace regroupe des structs de données pures (POD) sans logique ni héritage. Ils sont utilisés par l\textquotesingle{}\doxylink{classECS}{ECS} pour attacher des propriétés aux entités. Pas de relations directes entre eux, mais ils partagent un rôle commun dans l\textquotesingle{}architecture.
